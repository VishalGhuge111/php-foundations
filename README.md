# PHP Foundations

Learning backend development by building small projects and implementing core concepts from scratch.

## What's Inside

This repo contains my hands-on practice with PHP fundamentals:

- Form handling (GET/POST)
- MySQL database integration
- User authentication basics
- CRUD operations
- File uploads
- Session management

## Approach

Instead of just following tutorials, I'm learning by building. Each folder is a focused implementation or mini-project that solves a specific problem.

## Structure
/01-basics/           # PHP syntax, variables, functions
/02-forms-db/         # Form handling + MySQL connection
/03-auth-system/      # Basic login/signup
/04-crud-app/         # Create, Read, Update, Delete
/05-file-uploads/     # Basic file upload logic

## Tech Stack

- PHP (vanilla — no frameworks yet)
- MySQL
- HTML/CSS (minimal, just for functionality)

## Progress

Working through backend fundamentals step-by-step. Each implementation is documented with comments explaining the logic.
